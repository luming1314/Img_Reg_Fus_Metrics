# Image Registration and Fusion Metrics
## 配准指标
配准指标是参考论文[RFNet: Unsupervised Network for Mutually Reinforcing Multi-modal Image Registration and Fusion](https://doi.org/10.1109/CVPR52688.2022.01906)，自己实现的。

## 融合指标
融合指标python实现参考（唐霖峰博士）：https://blog.csdn.net/fovever_/article/details/129332278

融合指标matlab实现参考（唐霖峰博士）：https://github.com/Linfeng-Tang/Evaluation-for-Image-Fusion

对于唐霖峰博士实现的图像融合指标，可以参考如下论文：

  1. [Infrared and visible image fusion methods and applications: A survey](https://doi.org/10.1016/j.inffus.2018.02.004)
  
  2. [基于深度学习的图像融合方法综述](https://txtx.publish.founderss.cn/zh/article/doi/10.11834/jig.220422/)

此外，部分指标唐霖峰博士未实现的，可以参考如下论文：

  1. [Objective assessment of multiresolution image fusion algorithms for context enhancement in night vision: A comparative study](https://doi.org/10.1109/TPAMI.2011.109) 【[代码](https://github.com/yuliu316316/MFIF/tree/master)】
  
  2. [Rethinking the effectiveness of objective evaluation metrics in multi-focus image fusion: A statistic-based approach](https://doi.org/10.1109/TPAMI.2024.3367905) 【[代码](https://github.com/yuliu316316/MFIF-Metrics)】
